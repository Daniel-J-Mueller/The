# The color palette

The editor presentation uses five project colors and restrained pastel
derivatives. These are presentation tokens, not language semantics.

| Role | Color |
|---|---|
| Plum background | `#351332` |
| Slate structure | `#454758` |
| Teal comments | `#699A96` |
| Cream literals | `#EACBA3` |
| Crimson control | `#CC2038` |
| Pastel rose procedures | `#E05D70` |
| Pastel mauve PAGEs | `#C8A7C3` |
| Pastel aqua strings/types | `#9FC8C4` / `#A8CFCA` |
| Pastel green positions | `#8FBDA8` |
| Pastel orange `at` | `#D99A73` |

New syntax categories should first reuse an existing semantic color. Add a pastel
derivative only when adjacent categories would otherwise be difficult to
distinguish.
